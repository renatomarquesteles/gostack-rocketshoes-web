# rocketshoes
Aplicação desenvolvida em ReactJS, com arquitetura Flux, utilizando Redux, Redux Saga, Styled Components, Toastify, Reactotron e muito mais.

Esta aplicação simula um e-commerce, onde é possível adicionar e remover produtos no carrinho
